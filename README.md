# Fake-News-Detector-Java
Java based Fake News Detection system using Machine Learning.

##  Features
- Detects fake and real news
- Uses Naive Bayes classifier
- Indian news dataset
- Easy to run Java project

##  Tech Stack
- Java
- Machine Learning
- Weka
- NLP

##  How to Run
1. Add weka.jar to build path
2. Run Main.java
3. Enter news text
